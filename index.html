用户访谈的原则

发表于 2019-04-03 | 分类于 随笔
  1. “最近一次”原则：不要让用户干巴巴地描述观点和行为，请他们结合最近一次的场景体验去讲故事。把回忆交 给用户，把提炼和总结，交给我们。
  2. “鹦鹉学舌”原则：对于访谈的关键性问题，或是用户描述不清之处，通过重复用户的描述，可有效避免理解性 偏差，并适时总结提炼用户的观点。
  3. “沉默10秒”原则：抛出问题，或是展示方案时，给用户充分思考和回忆的时间，沉默之后，用户会给予更多。
  4. “不答反问”原则：当用户主动提出问题时，不要急于给出答案。变被动回答为主动追问，才可刨根问底，层层 探知用户本质需求。
  5. “转场衔接”原则：合理转场，用承上启下的句子衔接不同话题，让访谈生动而不生硬。

产业路由器

发表于 2019-04-03 | 分类于 随笔

为了让整个行业的效率更高，三维家不仅仅是一个云设计平台，而是提供了从前端到后端的四个工具：营销工具、设计工具、生产工具和交付工具。

营销工具帮助企业和经销商“以一张图为中心，把消费者和商家更好地联结在一起”，实现“内容找人”。例如2019年三维家推出“逛逛美家”，帮助商户通过内容+小工具+社群裂变，获得客流。

设计工具不仅仅是云设计平台，更是用AI工具实现让产品“自己会说话”。

而生产工具则是做到做全品类而不单是橱柜、衣柜、淋浴柜、家装部品化的智能制造，打通前段设计和后端生产，实现一键下单、实时生产。而交付工具，则是以工具为中心来指挥生产、送货、安装、交付，甚至实现施工的监控等。

第一性原理

发表于 2019-03-31 | 分类于 随笔

参考链接：https://www.jianshu.com/p/dc4e03edace2?from=timeline&isappinstalled=0

打扮家崔健：“十大品牌”受到老百姓欢迎，是因为降低了老百姓的选择成本；同样，一个公司是否具有巨大价值，一定要搞明白你能为供应链降低多少综合成本，这是第一性原理。
打扮家崔健：未来十年，VR/AR逐渐替代手机成为下一代计算平台，这和当年手机逐渐替代PC，是基于同一个第一性原理。站在这个角度看，近两年的VR/AR热度回落，不过是个小插曲而已。

第一性原理 first principle

物理学研究的第一性原理，不要以类比方式来推理，应该从最根本的真理开始思考。物理学家就是运用这种原则，搞懂一些反直觉的东西，例如量子力学。
我想说的是，把一切都归结于事物的基本本质然后从此处开始推理，这不同于类比推理。我们生活当中，我们都是用类比推理，实际上就是说我们要照搬别人的做法然后稍加改动。你必须这样做。否则，从心理层面看，你这一天就过不去了。不过当你想要做一件新的事情，你必须按物理原则办事。
多利用这个原则来判断，你做的事情是否真的合理有意义，还是其实你只是在跟随别人？如果你想开创新事业、开发新产品，这是最好的思考方法。

npm用法

发表于 2018-09-30 | 分类于 随笔

npm registry

搜索镜像: https://npm.taobao.org

  1. npm 可以使用taobao的镜像源,首先安装nrm

    npm install -g nrm

  2. 添加registry地址
    nrm add npm http://registry.npmjs.org
    nrm add taobao https://registry.npm.taobao.org
  3. 切换npm registry地址
    nrm use taobao
    nrm use npm

一些常用快捷键

  • 清除当前行：ctrl + u
  • 到行首：ctrl + a
  • 到行尾：ctrl + e
  • 前进后退：ctrl + f/b (相当于左右方向键)

分屏

  • 垂直分屏：command + d
  • 水平分屏：command + shift + d
  • 切换屏幕：command + option + 方向键 command + [ 或 command + ]
  • 查看历史命令：command + ;
  • 查看剪贴板历史：command + shift + h
